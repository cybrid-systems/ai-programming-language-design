## 📊 **支持的电路类型**

### **完整覆盖范围**
| 类别 | 具体类型 | 复杂度 | 验证级别 |
|------|----------|--------|----------|
| **线性电路** | RC/LC滤波器、分压器、运放电路 | 简单 | KCL/KVL + 参数范围 |
| **非线性电路** | 二极管整流、MOSFET放大器、BJT电路 | 中等 | Shockley方程 + 平方律模型 |
| **开关电源** | Buck/Boost转换器、同步整流 | 复杂 | 瞬态稳定性 + 效率约束 |
| **混合信号** | ADC前端、传感器放大、接口电路 | 中等 | 噪声分析 + 精度验证 |
| **高级系统** | 温度补偿、保护电路、控制环路 | 复杂 | 多约束联合验证 |
