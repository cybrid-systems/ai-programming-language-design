# 基础元件参考手册

## 🎯 **元件语法总览**

所有元件遵循统一语法：
```racket
(元件类型 元件标识 值/参数 (nodes 节点1 节点2 [节点3]) [#:选项 ...])
```

## 📋 **8种基础元件**

### **1. 电压源 (vsource)**
```racket
(vsource V1 5.0 (nodes 1 0))
```
- **值**: 电压值（伏特）
- **节点**: 2个节点（正极，负极）
- **SPICE对应**: `Vname n+ n- DC value`
- **示例**: `(vsource Vcc 12.0 (nodes 1 0))`

### **2. 电阻 (resistor)**
```racket
(resistor R1 1000 (nodes 1 2))
```
- **值**: 电阻值（欧姆）
- **节点**: 2个节点
- **SPICE对应**: `Rname n1 n2 value`
- **示例**: `(resistor Rload 220 (nodes out 0))`

### **3. 电容 (capacitor)**
```racket
(capacitor C1 1e-6 (nodes 2 0))
```
- **值**: 电容值（法拉）
- **节点**: 2个节点
- **SPICE对应**: `Cname n1 n2 value`
- **示例**: `(capacitor Cbypass 100e-6 (nodes vcc 0))`

### **4. 电感 (inductor)**
```racket
(inductor L1 10e-6 (nodes 2 3))
```
- **值**: 电感值（亨利）
- **节点**: 2个节点
- **SPICE对应**: `Lname n1 n2 value`
- **示例**: `(inductor Lfilter 22e-6 (nodes 1 2))`

### **5. 二极管 (diode)**
```racket
(diode D1 (nodes 3 4))
```
- **值**: 无（使用默认模型）
- **节点**: 2个节点（阳极，阴极）
- **SPICE对应**: `Dname anode cathode Ddefault`
- **示例**: `(diode Dclamp (nodes signal 0))`

### **6. 运算放大器 (opamp)**
```racket
(opamp U1 (nodes 4 0 5))
```
- **值**: 无
- **节点**: 3个节点（反相输入，同相输入，输出）
- **SPICE对应**: `Xname -in +in out opamp`
- **示例**: `(opamp U1 (nodes in- in+ out))`

### **7. 开关 (switch)**
```racket
(switch S1 (nodes 5 6))
```
- **值**: 无
- **节点**: 2个节点
- **SPICE对应**: `Sname n1 n2 n1 0 SW`
- **示例**: `(switch S1 (nodes vin vout))`

### **8. MOSFET (mosfet)**
```racket
(mosfet M1 (nodes 6 7 8) #:type nmos)
```
- **值**: 无
- **节点**: 3个节点（漏极，源极，栅极）
- **选项**: `#:type nmos` 或 `#:type pmos`
- **SPICE对应**: `Mname drain source gate 0 nmos/pmos`
- **示例**: `(mosfet M1 (nodes drain source gate) #:type pmos)`

## 🎨 **语法设计细节**

### **节点表示**
```racket
;; 数字节点（推荐）
(nodes 1 0)      ; 节点1到地
(nodes 1 2 3)    ; 三端器件

;; 符号节点（未来支持）
(nodes vin vout) ; 自动编号
```

### **值表示**
```racket
;; 直接数值
(resistor R1 1000 (nodes 1 2))

;; 科学计数法
(capacitor C1 1e-6 (nodes 2 0))

;; 带单位（未来支持）
(resistor R1 1k (nodes 1 2))   ; 1千欧
(capacitor C1 100n (nodes 2 0)) ; 100纳法
```

### **选项语法**
```racket
;; 目前仅MOSFET需要选项
(mosfet M1 (nodes d s g) #:type nmos)

;; 未来扩展
(resistor R1 1000 (nodes 1 2) #:tolerance 0.05)
(capacitor C1 1e-6 (nodes 2 0) #:voltage 16)
```

## 🔧 **验证规则**

### **通用规则**
1. **节点数检查**: 每个元件有正确的节点数
2. **参数范围**: 电阻、电容、电感值 > 0
3. **标识唯一性**: 元件标识不能重复
4. **接地检查**: 建议包含节点0（地）

### **元件特定规则**
| 元件 | 节点数 | 特殊要求 |
|------|--------|----------|
| **电压源** | 2 | 电压值可为正负 |
| **电阻** | 2 | 值 > 0 |
| **电容** | 2 | 值 > 0 |
| **电感** | 2 | 值 > 0 |
| **二极管** | 2 | 无 |
| **运算放大器** | 3 | 必须3个节点 |
| **开关** | 2 | 无 |
| **MOSFET** | 3 | 必须指定 #:type |

## 📊 **典型电路示例**

### **RC低通滤波器**
```racket
(define-circuit rc-lowpass
  #:title "一阶RC低通滤波器"
  #:analysis (transient #:stop-time 10e-3)
  
  (vsource V1 5.0 (nodes 1 0))
  (resistor R1 1000 (nodes 1 2))
  (capacitor C1 1e-6 (nodes 2 0))
  
  (probe voltage Vout 2))
```

### **二极管整流器**
```racket
(define-circuit diode-rectifier
  #:title "二极管半波整流器"
  #:analysis (transient #:stop-time 20e-3)
  
  (vsource Vac 12 (nodes 1 0))
  (diode D1 (nodes 1 2))
  (capacitor C1 100e-6 (nodes 2 0))
  (resistor Rload 100 (nodes 2 0))
  
  (probe voltage Vrect 2))
```

### **运算放大器反相放大器**
```racket
(define-circuit opamp-inverting
  #:title "反相运算放大器"
  #:analysis (dc)
  
  (vsource Vin 1.0 (nodes 1 0))
  (resistor Rin 1000 (nodes 1 2))
  (resistor Rf 10000 (nodes 2 3))
  (opamp U1 (nodes 2 0 3))
  
  (probe voltage Vout 3))
```

### **MOSFET开关电路**
```racket
(define-circuit mosfet-switch
  #:title "MOSFET开关电路"
  #:analysis (dc)
  
  (vsource Vdd 12 (nodes 1 0))
  (resistor Rload 100 (nodes 1 2))
  (mosfet M1 (nodes 2 0 3) #:type nmos)
  (vsource Vgs 5 (nodes 3 0))
  
  (probe voltage Vdrain 2))
```

## 🔍 **常见问题**

### **Q: 节点编号有什么规则？**
**A**: 建议从1开始编号，0保留为地。节点编号不需要连续，但建议保持有序。

### **Q: 如何表示负电压？**
**A**: 使用负值：`(vsource Vneg -5.0 (nodes 1 0))`

### **Q: 如何表示大电阻/电容值？**
**A**: 使用科学计数法：
- `1e6` = 1兆欧
- `1e-9` = 1纳法
- `1e-12` = 1皮法

### **Q: MOSFET的节点顺序是什么？**
**A**: `(nodes 漏极 源极 栅极)`，例如：`(nodes drain source gate)`

### **Q: 运算放大器的节点顺序是什么？**
**A**: `(nodes 反相输入 同相输入 输出)`，例如：`(nodes in- in+ out)`

## 📈 **扩展计划**

### **短期扩展**
1. **更多元件**: BJT、变压器、晶振
2. **参数化值**: `(* 1k R)`、`(/ 1 (* 2 pi f C))`
3. **命名节点**: `#:nodes ((vin 1) (vout 2) (gnd 0))`

### **中期扩展**
1. **子电路**: `(define-subcircuit rc-filter ...)`
2. **参数扫描**: `#:sweep (R (100 10000 100))`
3. **约束条件**: `#:constraint (voltage Vout > 2.5 < 3.3)`

### **长期扩展**
1. **概率类型**: `#:tolerance 0.05`、`#:distribution normal`
2. **优化目标**: `#:optimize (minimize power)`
3. **布局提示**: `#:pcb-hint "power-compact"`
