# OceanBase 源码分析 31-40 篇规划

## 已完成 30 篇结构

**01-10：** MVCC 行 → Iterator → 写写冲突 → 回调链 → Compact → Freeze → LS Tree → SSTable → SQL DAS → 2PC
**11-20：** PALF → Election → Clog → Memtable 内部 → KeyBtree → Conflict Handler → Optimizer → 索引 → 分区迁移 → 备份恢复
**21-30：** PX → Plan Cache → Parser/Resolver → 类型系统 → 内存管理 → 编码引擎 → RootServer → GTS → SQL 诊断 → OBServer 启动
