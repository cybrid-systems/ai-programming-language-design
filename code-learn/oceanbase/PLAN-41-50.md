# OceanBase 源码分析 41-50 篇规划

## 已完成 40 篇结构

**01-10：** MVCC Row → Iterator → Write Conflict → Callback → Compact → Freeze → LS Tree → SSTable → DAS → 2PC
**11-20：** PALF → Election → Clog → Memtable 内部 → KeyBtree → Conflict → Optimizer → 索引 → 迁移 → 备份
**21-30：** PX → PlanCache → Parser → 类型系统 → 内存 → 编码引擎 → RootServer → GTS → SQL 诊断 → 启动
**31-40：** DML → 表达式 → 子查询 → Merge → MacroBlock → 并发控制 → 路由 → 成员变更 → 租户 → MySQL 协议
