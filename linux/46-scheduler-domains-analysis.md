## 0. 概述

调度器域：CPU 拓扑层次组织支持负载均衡。实现在 kernel/sched/topology.c。

```c
// scheduler-domains 核心结构
struct scheduler_domains_data {
    void *private;
    unsigned long flags;
    struct list_head list;
```
